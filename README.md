roine.github.com
================

My repositories list into a cool UI

[roine.github.com/repositories](http://roine.github.com/repositories)

Current features:
  - Sorting
  	- created
  	- updated
  	- name
  	- language
  - Details
  	- name
  	- created
  	- updated
  	- description
  - Display a custom user's repositories
